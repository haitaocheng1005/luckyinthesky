# Lucy in the Sky — website concept demo

A high-impact, responsive one-page concept for the independent Liverpool cafe **Lucy in the Sky**.

Live demo: https://haitaocheng1005.github.io/luckyinthesky/

## Run locally

```powershell
pnpm.cmd install
pnpm.cmd dev
```

Production build:

```powershell
pnpm.cmd build
```

## Notes

- Business details and brand direction were researched on 15 July 2026.
- The food and interior imagery in `public/assets/` is original concept imagery generated for this demo; it does not copy Instagram photographs.
- Seasonal items and opening details should be reconfirmed before publishing a real site.
